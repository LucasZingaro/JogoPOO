Lista de telas do jogo{
    FrmStart == tela de inicio Carregamento de jogo  ou Novo Jogo:
        |->Novo Jogo == Ir para FrmNovoJogo;
        |->Carregar Jogo == Ir para Frm;
        FrmCarregarJogo == Tela de carregamento de jogos salvos:
            Lista de Jogos Salvos->Carregar == Ir para FrmTelaPrincipal;
            |->Voltar == Volta para a tela de inicio FrmStart;
        FrmNovoJogo-> Novo jogo, dado o nome do jogador:
            FrmIntroducao -> Tela de Explicação do jogo;
                |-> Pular == Ir para Tela Principal
                FrmTelaPrincipal == Tela de monitoramento, com ações do jogo:
                    |->Passar de Turno;
                    |->Salvar o jogo;
                    |->Pegar Emprestimo;
                    |->Tabela da Bolsa de Valores;
                    FrmDetalhesAcao == Detathes da ação:
                        |-> mais irformações sobre a ação
                        |-> Ordens de Compra e Venda
                        |-> Vender e comprar ações
}
